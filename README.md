# Real-Time Chat System (Node.js + TypeScript + Socket.IO)

This is a real-time chat backend with:
- JWT authentication
- Room-based messaging
- Express server
- Socket.IO real-time engine
- Frontend demo (located in /public)
- Support for scaling (Redis ready)
